# Streamly Backend API

A backend system for a video-sharing platform built using Node.js, Express.js, and MongoDB. The API supports video uploads, authentication, comments, playlists, and cloud-based media storage.

---

## Features

- JWT Authentication & Authorization
- Video Upload & Management
- Comment System
- Playlist Management
- Cloudinary Media Storage
- Multer File Upload Handling
- RESTful APIs
- MongoDB Database Integration
- Secure Route Protection

---

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Multer
- Cloudinary
